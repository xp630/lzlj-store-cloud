# LZLJ Cloud 泸州老窖云店系统 - 开发路线图

## 项目概述

基于 **Spring Cloud Alibaba** 的云原生微服务架构，服务于泸州老窖云店业务系统。

## 开发规范

### 分支管理
```
main        - 主分支 (生产)
develop     - 开发分支
feature/*   - 功能分支
bugfix/*    - 修复分支
release/*   - 发布分支
```

### 提交规范
```
feat:     新功能
fix:      修复bug
docs:     文档更新
style:    代码格式
refactor: 重构
test:     测试
chore:    构建/工具
```

### Code Review检查点
- [ ] 接口设计合理性
- [ ] 异常处理完整性
- [ ] 日志规范
- [ ] 安全性 (SQL注入、XSS等)
- [ ] 性能考虑 (N+1查询等)
